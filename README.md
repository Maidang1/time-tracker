# Chrono Pulse

Chrono Pulse 是一个使用 Taro + React 构建的跨端事件记录器。它可以在微信小程序、H5 等多端运行，帮助用户跟踪事件与每个记录的时间投入。

## 本地开发

```bash
pnpm install
pnpm run dev:weapp # 或 dev:h5
```
